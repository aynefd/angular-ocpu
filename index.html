OpenCPU AngularJS App Demo

The sole purpose of these examples is to demonstrate how to use the angular-ocpu.js library to call R from AngularJS applications using OpenCPU.

Introduction

The angular-ocpu.js library depends on angular-ocpu.js and provides a promise based OpenCPU API as the $ocpu service. A copy of angular.js and angular-ocpu.js must be included with your OpenCPU app.

Topics

Each topic consists of a minimal HTML page demonstrating basic functionality. Look at the source code of the HTML page to learn what is happening.
